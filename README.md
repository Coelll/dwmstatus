# dwmstatus
Dwm status bar.

![screenshot](screenshot.png)

Some of the shortcuts.
----------------------
```
super + F12 - volume up,
super + F11 - volume down,
super + F9  - mute the volume.
```

Installation.
-------------

```
git clone https://github.com/Coelll/dwmstatus
cd dwmstatus
sudo make install
```
